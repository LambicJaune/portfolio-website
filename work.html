my TomHanksApp / React-based

myTomHanksApp is a full-stack movie web application that allows users to browse and explore movies featuring Tom Hanks.

- Features include user registration, login, secure token-based authentication, movie filtering, favorites management, and user profile updates.

An alternative version of the frontend of myTomHanksApp, using the Angular framework and TypeScript.

- Uses dialogs to display data from a movie card, e.g. synopsis or a director's biography

Small-Talk

A mobile chat application built with React Native and Expo.

It provides users with a seamless chat experience where they can send messages, images taken from their camera or gallery, or display their location.

The app uses Google Firebase for real-time data storage and authentication.

Check it out!

The Pokédex

The Pokédex is a simple application that allows users to load dynamically a list of items (Pokémon) as well as additional information regarding each of them on demand (user action).

Data is fetched from an external public API called PokéAPI.

It is built solely using JavaScript, HTML and CSS + Bootstrap.



Check it out!

The MEET App

Meet is a serverless, progressive web application (PWA) built with React, using a test-driven development (TDD) approach. The app uses the Google Calendar API to fetch and display upcoming tech events in their city. It supports offline use, is installable as a mobile or desktop app, and provides powerful filtering and data visualization charts.
