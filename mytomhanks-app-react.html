Objective

Build a secure, scalable app using modern full-stack technologies.

Role: Full-stack developer

Duration: 2-3 weeks

Tech: React, Redux, Node.js, Express, MongoDB, Bootstrap, Netlify

The Challenge

How can users securely browse, filter, and personalize a movie collection while maintaining a clean, responsive interface and scalable backend ?

The Process

Backend & API

- Built a RESTful API with Node.js and Express

- Managed movie and user data with MongoDB Atlas and Mongoose

- Implemented secure authentication and authorization using JWT tokens and Passport.js

- Enabled cross-origin requests with CORS

- Deployed backend on Heroku

- Designed endpoints for users, movies, and favorites, then tested them with Postman

Challenge: Securing protected routes

Solution: JWT authentication and middleware

Frontend & State Management

- Built the user interface with React and React Router

- Implemented global state management with Redux

- Added filtering by title, genre, and director

- Deployed frontend on Netlify

Challenge: Managing shared application state

Solution: Centralized Redux store

Final Solution

The final application allows users to:

- Register and log in securely

- Browse and filter movies

- Save favorites

- Update their profile

Results & Learnings

- Delivered a complete full-stack application

- Gained experience with authentication, authorization, and state management

- Learned the importance of planning for scalability early

Looking back:

I was comfortable creating views for registration, login, movie overview,

and user profiles, adding features like filters and favourites, which gave

me confidence handling React state and feature complexity. Experimenting

with different filter types allowed me to try multiple approaches, though

filtering by director with a fixed-card layout created limitations. I

addressed these by simplifying features and reflecting on value, improving

usability, maintainability, and overall code quality.

Next steps: Improve accessibility, expand the movie database, and add testing
