MYTOMHANKS APP

Objective

Create a full-stack movie web application that allows user to browse and explore movies featuring Tom Hanks. The application includes a backend API, as well as a React-based frontend client that connects to the API and provides a rich, responsive user interface.

Key features:

- A RESTful backend API built with Node.js, Express, and MongoDB.

- User registration and login

- Secure token-based authentication

- Movie filtering using a search bar, or a selection of filters by genre or director

- Favorites management

- User profile updates

Technologies Used:

- React - For a dynamic user interface

- React Router - For seamless navigation with dynamic URLs

- Redux - For state management

- Bootstrap 5.3.7 and minimized CSS - For responsive design and optimized loading times

- Parcel - For bundling and optimizing assets

- Netlify - For hosting the application
