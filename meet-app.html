THE MEET APP

Objective

To create a user-friendly event planner that sources events from the Google Calendar API, allowing event filtering by city and number, and to visualize data interactively. The app is fully responsive and supports offline functionality.

Key features:

- Google Calendar OAuth integration.

- Dynamic event list based on city and number of events.

- Offline support via service workers.

- LocalStorage caching of event data.

- Fully tested with Jest, Testing Library, and BDD with Cucumber.

Technologies Used:

- React.js

- Recharts

- Google Calendar API

- Jest & React Testing Library

- Cucumber (BDD)

- Vite

- AWS Lambda (Serverless Functions)

- Git & Github Pages
