THE POKÉDEX APP

Objective

To build a small interractive JavaScript application that lets users explore a wide range of Pokémon, using HTML, CSS and JavaScript. Using the PokéAPI, the app fetches and displays each Pokémon's image as well as details regarding their height and types. Users can search for specific Pokémon by name, and navigate through the list using pagination controls. The app is designed to be responsive and user-friendly, providing an engaging experience for Pokémon enthusiasts.

Key features:

- Loads data from an external API

- Displays a list of items

- On user action (e.e. by clicking on a list item), displays details for that item

- Search-as-you-type filter to retrieve Pokémon by name

- Responsive design for various screen sizes

Technologies Used:

- JavaScript (ES6) - Provides the app’s logic, including DOM manipulation, API requests, and event handling.

- HTML 5 & CSS3 - Structure (HTML) and styling (CSS), with CSS3 features like gradients, transforms, media queries, etc.

- ESLint rules - To ensure code quality and consistency

- Bootstrap 5.3 - To facilitate responsive design and layout

- GitHub Pages - To host the application

Reflection on this project :

While this is a very basic project, this was my very first introduction to coding and was very exciting. I learned a lot about how to structure a web application, work with APIs, and implement interactive features using JavaScript. It was rewarding to see my ideas come to life and to create something that others could use and enjoy.
