THE SMALL TALK APP

Objective

To build a chat app for mobile devices using React Native. The app provides users with a chat interface and options to share images taken in real-time or from their gallery, as well as display their location. It supports offline functionality using their cache to display the content from their previous session.

Key features:

- Anonymous login

- Personalization of the user name and background color

- Real-time messaging

- Offline storage allowing access to previous messages when network is out of reach, and direct loading of new messages when network is back

- Sending images and location sharing

Technologies Used:

- React Native - A framework for cross-platform mobile development

- Expo - A platform simplifying the creation of universal native apps by providing libraries, tools and testing

- Cloud Firestore from Firebase - A NoSQL cloud database for storing and syncing data in real-time

- Gifted Chat - A library for building chat UI in React Native

- AsyncStorage - A simple, unencrypted, asynchronous, persistent, key-value storage system that is global to the app
