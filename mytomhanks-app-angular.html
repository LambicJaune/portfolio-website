MYTOMHANKS APP

Objective

Create a full-stack movie web application that allows user to browse and explore movies featuring Tom Hanks. The application includes a backend API, as well as an Angular-based frontend client that connects to the API and provides a rich, responsive user interface.

Key features:

- A RESTful backend API built with Node.js, Express, and MongoDB.

- User registration and login

- Secure token-based authentication

- Movie cards offering the possibility to filter movies by genre, as well as open a dialog displaying their synopsis or their director's biography

- Favorites management

- User profile updates

Technologies Used:

- Angular 19.1.0 - For a dynamic user interface

- Angular Material - For reusable components and a consistent, responsive design with built-in accessibility

- TypeScript - For maintaining the code

- RxJS - For reactive programming and handling asynchronous data streams

- TypeDoc - For generating documentation from the codebase
