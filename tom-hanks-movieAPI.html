The Tom Hanks Movie API

Objective

To build the server-side component of the "myTomHanks App" movie application. The web application provides users with access to information about different movies featuring the actor, including their directors, and genres. Users can sign up, update their personal information, and create a manageable list of their favorite movies.

Technologies Used:

- Node.js & Express - For building the RESTful API and handling API requests

- MongoDB Atlas & Mongoose - For cloud-based storing and managing movie/user data

- JWT tokens & Passport.js - For secure authentication and authorization

- CORS - For enabling cross-origin requests

- Heroku - For backend deployment
